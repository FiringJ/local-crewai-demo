| 文件 | 用途 |

| `contract_audit_rules.md` | 26 项固定检查规则明细（R01–R26） |
| `contract_audit_framework.md` | 合同类型与上位法对照框架 |
| `realistic_software_service_contract.txt` | 128 万元脱敏样本合同（附件测试用） |
| `sample_contract.txt` | 短样本合同 |
| `review-response.json` | 样本合同的 26 项规则命中结果（16 通过 / 7 未通过 / 3 需复核） |
