# 企业合同初审知识库（办公小浣熊 @知识库）

## 适用场景

采购合同、技术服务合同、软件实施合同的标准化初审。供审核 Agent 在语义判断时引用，不得覆盖规则引擎的确定性计算结论。

## 高风险红线（必须拦截）

1. **争议解决冲突**：同时约定仲裁与诉讼，或管辖约定互相矛盾。
2. **金额三件套不一致**：大写金额、数字金额、明细合计三者任一不符。
3. **税务计算错误**：不含税金额 × 税率 ≠ 税额，或含税总额不等。
4. **付款比例**：分期付款比例合计不等于 100%。
5. **主体不一致**：首部、签章处、正文中的甲乙双方名称不一致。

## 中风险关注项

- 付款条件仅写「验收后付款」但未定义验收标准。
- 逾期违约金日费率超过 0.05%（万分之五）。
- 保密义务期限表述为「永久」或缺少例外情形。
- 知识产权归属约定模糊（背景 IP vs 交付成果）。

## 法务修改话术模板

| 问题类型 | 建议表述 |
| --- | --- |
| 争议解决 | 统一为「提交北京仲裁委员会仲裁，仲裁裁决为终局」 |
| 付款节点 | 明确「签署后 X 日内 / 验收合格后 X 个工作日内」 |
| 违约金上限 | 增加「累计不超过合同总价的 20%」 |
| 保密期限 | 改为「合作期间及终止后 3 年，法定保密信息除外」 |

## 行业税率参考（中国大陆）

- 软件/技术服务：通常 6%（现代服务）或 13%（含硬件）
- 建筑服务：9%
- 小规模纳税人：1% 或 3%

## 审核流程（OPC 一人全栈）

1. 上传合同 → 文档解析抽取字段
2. 规则引擎 22 项确定性校验
3. 知识库辅助语义复核
4. 数据分析输出合规率与风险分布
5. SenseChat 生成审核报告与汇报大纲
